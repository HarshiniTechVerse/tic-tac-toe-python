# 🎮 Tic Tac Toe using Python

## 📌 Project Overview
This is a simple Tic Tac Toe game developed using Python. The game allows two players to take turns and play on a 3x3 grid.

## 💡 Features
- Two-player gameplay
- Interactive console-based interface
- Win and draw detection
- Beginner-friendly Python project

## 🛠️ Technologies Used
- Python

## 🚀 How to Run the Project
1. Open `tic_tac_toe.py` in IDLE.
2. Press F5 or select Run → Run Module.
